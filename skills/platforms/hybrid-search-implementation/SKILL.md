# Hybrid Search Implementation

Use this skill for hybrid search implementation tasks where platform-specific implementation, architecture, or operational guidance is needed.

## When to Use

- Building RAG systems with improved recall.
- Combining semantic understanding with exact matching.
- Handling queries with specific terms (names, codes).
- Improving search for domain-specific vocabulary.

## Workflow

1. Confirm why vector-only or keyword-only retrieval is insufficient.
2. Design candidate generation, score fusion, filtering, and reranking as separate stages.
3. Recommend evaluation queries and metrics before tuning weights.

- Prefer current official documentation for version-sensitive APIs, commands, deployment options, and configuration names.
