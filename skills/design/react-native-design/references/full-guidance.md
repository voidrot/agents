# React Native Design Full Guidance

This reference preserves the long-form guidance layer required by the update plan while the active `SKILL.md` remains a concise routing entrypoint. Use the more focused companion references for deeper examples and detailed procedures.

## Detailed guidance

- Start by confirming React Native, navigation, gesture, and animation library versions and platform targets.
- Build layouts around safe areas, keyboard behavior, density, platform conventions, and accessibility labels/actions.
- Use navigation, gesture, and Reanimated patterns only when they clarify flow or feedback; keep reduced-motion and performance in scope.
- Test on iOS and Android paths for gestures, screen readers, layout shifts, and common low-end performance constraints.
