# Android Mobile Design Full Guidance

This reference preserves the long-form guidance layer required by the update plan while the active `SKILL.md` remains a concise routing entrypoint. Use the more focused companion references for deeper examples and detailed procedures.

## Detailed guidance

- Prefer Material Design 3, Jetpack Compose, dynamic color, and Android platform interaction conventions.
- Plan for multiple window sizes, foldables, tablets, text scaling, dark theme, configuration changes, and edge-to-edge layouts.
- Compose component guidance should include state, semantics, focus order, touch targets, navigation, and theme token use.
- Validate TalkBack, keyboard/switch paths, contrast, responsive layout classes, and device posture changes.
